# GameBox - Website Design Workflow Documentation

## Project Overview

**Website:** GameBox (gameboxsolutions.de)
**Platform:** WordPress + WooCommerce
**Business:** Curated themed gift boxes for kids (composite/bundle products)
**Product Format:** Composite products with required base items + optional selectable add-ons
**Target Audience:** Parents, relatives, gift buyers looking for themed children's gift boxes
**Languages:** English, German, French
**Currencies:** USD, EUR, JPY

## Product Structure (Composite Format)

Each themed box follows this structure:
1. **Base/Required Items** - Core themed toys (e.g., Building Blocks sets) - included in base price
2. **Included Accessories** - Colouring pencils, sharpener, marker pens, activity books - always included
3. **Optional Extras** - Selectable add-ons at extra cost:
   - Extra games/riddles ($2.00-$2.50)
   - Snack packs ($1.00)
   - Pocket money/cashback vouchers ($7.00-$18.50)
4. **Greeting Card** - Personalizable card with message (multiple language options)

**Base Price:** From $15.00 per box

## Design Direction

### Color Palette (Kid-friendly, Fun, Trustworthy)
- **Primary:** Bright Orange (#FF6B35) - energy, fun, playfulness
- **Secondary:** Deep Blue (#1A3A5C) - trust, reliability
- **Accent:** Lime Green (#7BC74D) - freshness, go/action
- **Background:** Light Cream (#FFF8F0) - warm, soft
- **Text:** Dark Charcoal (#2D2D2D) - readability
- **White:** #FFFFFF - clean spaces

### Typography
- **Headings:** Bold, rounded sans-serif (e.g., Nunito, Baloo 2, or Fredoka One)
- **Body Text:** Clean sans-serif (e.g., Open Sans, Poppins, or Nunito)
- **Accent/Fun Text:** Handwritten style for callouts (e.g., Caveat or Patrick Hand)

### Design Style
- Playful but professional
- Rounded corners on cards and buttons
- Soft shadows for depth
- Generous whitespace
- Large product images
- Icon-heavy for quick scanning
- Bright colored CTAs
- No mobile responsive needed (desktop-only focus)

### Layout Grid
- Max width: 1440px
- Content width: 1200px
- 12-column grid
- Gutter: 30px
